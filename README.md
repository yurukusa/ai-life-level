# AI Life Level

**あなたのAI生活、ゲームで言うと何難易度？**

7つの質問に答えるだけで、あなたのAI生活の「難易度」を判定。Easy Mode？それともHardcore？

**[診断する](https://yurukusa.github.io/ai-life-level/)**

## How It Works

1. 7つの質問に答える（所要時間: 1分）
2. AI依存度・課金力・生活変化などを分析
3. 5段階の難易度で判定
4. 結果をシェア

## Difficulty Levels

| Level | Score | Description |
|-------|-------|-------------|
| Easy Mode | 0-5 | まだチュートリアル。健全 |
| Normal Mode | 6-12 | バランスよく使いこなしている |
| Hard Mode | 13-19 | AIなしに戻れなくなりつつある |
| Nightmare Mode | 20-25 | AIと共依存。深夜作業が日常 |
| Hardcore Mode | 26-32 | 人生がハードコアゲームになった |

## Features

- Bilingual: Japanese / English
- 7 questions across 7 categories
- Score breakdown with visual bars
- Share on X / Copy result
- Dark gaming aesthetic
